# Absensi MTS Admin Web - School Attendance Management System

Absensi MTS Admin Web adalah aplikasi berbasis website yang digunakan untuk membantu proses pengelolaan data absensi siswa secara digital pada lingkungan sekolah MTS.

Aplikasi ini digunakan oleh pihak admin sekolah untuk mengelola data utama seperti data siswa, guru, kelas, dan rekapitulasi absensi.


## About Project

Project ini merupakan bagian dari Sistem Informasi Absensi MTS yang terdiri dari aplikasi admin berbasis web dan aplikasi mobile.

Admin Web digunakan sebagai pusat pengelolaan data agar proses administrasi sekolah menjadi lebih efektif, terstruktur, dan mudah diakses.

Data yang dikelola melalui aplikasi admin akan digunakan pada aplikasi mobile oleh guru dan siswa.

## Installation

Clone repository

```bash
git clone https://github.com/Sheva1012/absensi-mts-admin-web.git
```

Masuk folder project

```bash
cd absensi-mts-admin-web
```

Install dependency

```bash
flutter pub get
```

Konfigurasi environment

```env
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
```

Menjalankan aplikasi

```bash
flutter run
```
